📡 COS'È QSLFORGE

QSLforge è un generatore gratuito di QSL card per radioamatori. Importa il tuo log in formato ADIF, posiziona i campi sull'immagine QSL e genera le card pronte da stampare o inviare digitalmente.

🔒 Privacy garantita: tutto viene elaborato localmente nel tuo browser. Nessun dato viene inviato a server esterni. Il log e le immagini non lasciano mai il tuo dispositivo.

🔧 COME SI USA

  • Step 1 — Setup QSL: carica l'immagine della tua QSL card (JPG, PNG, WebP). Clicca i campi dall'elenco per aggiungerli, trascinali nella posizione desiderata. Per ogni campo puoi impostare font, dimensione, colore, grassetto, ombra e allineamento.
  • Step 2 — Importa log: carica il file .adi o .adif. Seleziona i QSO per cui vuoi generare le card usando le checkbox.
  • Step 3 — Genera: clicca "Genera tutte" per creare le QSL. Scarica le immagini singolarmente o tutte in un archivio ZIP.

📋 CAMPI ADIF SUPPORTATI

CALL QSO_DATE TIME_ON BAND FREQ MODE RST_SENT RST_RCVD NAME QTH GRIDSQUARE DXCC TX_PWR COMMENT OPERATOR IOTA CQZ ITUZ STATE CONT POTA_REF SOTA_REF WWFF_REF

La data viene formattata come DD/MM/YYYY, l'ora come HH:MM.

💾 SALVATAGGIO CONFIGURAZIONE

Posizioni e impostazioni di ogni campo vengono salvate automaticamente nel browser (localStorage) e ripristinate alla prossima apertura. Puoi anche esportare la configurazione in un file .json e reimportarla su un altro dispositivo o browser tramite i pulsanti nella scheda Setup.

🌍 LINGUE

QSLforge supporta la localizzazione tramite file JSON nella cartella locales/. Sono inclusi italiano (it.json) e inglese (en.json). Per aggiungere una lingua crea locales/xx.json con le stesse chiavi tradotte, poi imposta localStorage.setItem('qslforge_lang','xx') nella console del browser.
